# 🚀 ExamNova – Multi-Tenant SaaS Online Examination Platform

ExamNova is a full-stack MERN application that allows multiple teachers to create and manage online examinations for their respective students through a secure and isolated environment.

## 🌟 Key Features

- JWT-based Authentication
- Role-Based Access Control (Teacher & Student)
- Multi-Tenant Architecture
- Timed Online Exams
- Automated Evaluation
- Analytics Dashboard
- Secure Data Isolation

## 🏗️ Tech Stack

Frontend: React.js  
Backend: Node.js, Express.js  
Database: MongoDB Atlas  
Authentication: JWT + Bcrypt  
Deployment: Vercel & Render
